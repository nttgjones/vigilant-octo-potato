Speechmatics Transcription and Translation

This script uses the Speechmatics API to transcribe and translate an audio file.

        
            # Example usage:
            git clone https://github.com/nttgjones/vigilant-octo-potato/blob/71266be25182099fbe54d7f408b7abc134127dd8/langtest
            cd speechmatics-transcription
            pip install -r requirements.txt
            python transcribe_translate.py

Replace YOUR_API_KEY and example.wav with your actual API key and audio file path.
